🎮 **Tank Game**

A simple multiplayer tank battle game built using Python and Pygame. Players control tanks, shoot bullets, and try to knock opponents off the map to win.

🚀** Features**
3-player gameplay
Tank movement in all directions
Shooting mechanics
Collision detection
Explosion effects
Win condition (push opponents off the cliff)
🎮** Controls**
Player 1
Move: W A S D
Shoot: SPACE
Player 2
Move: Arrow Keys
Shoot: ENTER
Player 3
Move: I J K L
Shoot: U
🖥️** How to Run**

Install Pygame:

pip install pygame

Run the game:

python tank_game.py

🧠 **How It Works**
Each player controls a tank using different keys
Players shoot bullets upward
When a bullet hits a tank:
The tank gets pushed (knockback)
An explosion animation is triggered
If a tank falls beyond the cliff → game over
